Nomial is a TypeScript implementation of High-Performance Polynomial Root Finding for Graphics (Yuksel 2022). A common application for root finding in graphics and CAD is intersections. This example shows how Nomial can be used to find the intersection of 100 cubic (degree 3) bezier curves. Each pairwise intersection requires finding the root of one degree 9 and two degree 3 polynomials.
